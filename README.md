# MONKEY-TOUCH

Monkey Touch is a clean, minimal sliding‑puzzle brain game built using pure HTML, CSS, and JavaScript.
The objective is to arrange the numbered tiles in the correct order by sliding them into the empty space.

This project focuses on simplicity, performance, and mobile‑friendly design while delivering a polished game experience directly in the browser.

---

## 🎮 Game Features

* 3 × 3 sliding puzzle grid
* One empty tile for movement
* Level progression system
* Timer and move counter
* Solvable random shuffle logic
* Mobile‑friendly and responsive layout
* Clean UI with subtle 3D tile effects
* No external libraries or frameworks

---

## 🧠 How to Play

1. The board starts with 8 numbered tiles and 1 empty space.
2. Click or tap a tile that is **adjacent to the empty space** to slide it.
3. Arrange the tiles in order from **1 to 8** with the empty space at the end.
4. When solved, the next level starts automatically.

---

## 🛠️ Tech Stack

* **HTML5** – Structure and layout
* **CSS3** – Styling, shadows, and responsive UI
* **JavaScript (Vanilla)** – Game logic, timer, shuffle, and interactions

No external libraries or dependencies are used.

---

## 🚀 Run Locally

1. Clone or download this repository.
